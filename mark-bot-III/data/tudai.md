# TUDAI - TECNICATURA UNIVERSITARIA EN DESARROLLO DE APLICACIONES INFORMATICAS

**Nombre de la carrera:** Tecnicatura Universitaria en Desarrollo de Aplicaciones Informáticas
**Duración:**
- 5 cuatrimestres para Tandil.
- 3 años para el resto de las sedes (Lobería y Las Flores).
**Sedes:**
- Tandil, provincia de Buenos Aires, Argentina (sede principal).
- Lobería, provincia de Buenos Aires, Argentina.
- Las Flores, provincia de Buenos Aires, Argentina.

## Plan de Estudios

### Primer Año
**1º Cuatrimestre**
- Programación 1
- Taller de Matemática Computacional
- Inglés 1

**2º Cuatrimestre**
- Tecnología de la Información en las Organizaciones
- Web 1
- Programación 2
- Inglés 2
- Seminario Tecnológico 1

### Segundo Año
**1º Cuatrimestre**
- Web 2
- Programación 3
- Introducción a las Metodologías de Desarrollo de Software
- Bases de Datos
- Seminario Tecnológico 2

**2º Cuatrimestre**
- Técnicas de Documentación y Validación
- Interfaces de Usuario e Interacción
- Arquitecturas Web
- Configuración y Desarrollo de Aplicaciones en Redes
- Seminario Tecnológico 3

### Tercer Año
**1º Cuatrimestre**
- Prácticas Profesionales (PPS)

## Ingreso

**Fechas de Ingreso:**
- **Instancia Virtual:** Septiembre - Noviembre.
- **Instancia Presencial:** Febrero - Marzo.
- **Instancia Semipresencial:** Abril - Junio (Exclusiva para recursantes).

**Materias del curso de ingreso:**
- **Fundamentos de la programación:** pauta de aprobación: parcial o recuperatorio con nota >= 4.
- **Introducción a la vida universitaria:** pauta de aprobación: trabajo práctico integrador.
